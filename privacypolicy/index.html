プライバシーポリシー

最終更新日：2026年4月18日

Teppei Uomi（以下「当方」）は、モバイルアプリケーション「X-Biz」（以下「本アプリ」）における利用者の個人情報について、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 取得する情報

本アプリは、サービス提供のため以下の情報を取得します。

  • アカウント情報：Sign in with Apple により提供されるユーザー識別子（Apple ID に紐づく匿名ID）、メールアドレス（利用者がリレー／実アドレスいずれかを選択）、表示名
  • 利用ログ：業務記録（カテゴリ、開始・終了時刻、所要時間）、時給単価、AIコーチとの会話履歴、プラン状態（Free/Pro/トライアル）
  • 端末情報：OS種別・バージョン、アプリバージョン、広告識別子（ATT許諾時のみ）
  • クラッシュ／診断情報：アプリの安定稼働のため匿名のクラッシュログ

2. 利用目的

  • サービスの提供・維持・改善
  • 本人確認・認証（Sign in with Apple）
  • AIコーチによる分析・レビューの生成
  • 不具合調査および品質改善
  • 広告の配信および効果測定（Google AdMob）
  • 法令に基づく対応

3. 第三者提供・委託先

本アプリは以下のサービスを利用しており、各社のプライバシーポリシーが適用されます。

  • Supabase, Inc.（認証およびデータ保存）
    https://supabase.com/privacy
  • Apple Inc.（Sign in with Apple、App Store決済管理）
    https://www.apple.com/legal/privacy/
  • RevenueCat, Inc.（サブスクリプション状態の管理）
    https://www.revenuecat.com/privacy
  • Google LLC / Google AdMob（広告配信）
    https://policies.google.com/privacy
  • Rork / OpenAI 等の生成AI API プロバイダ（AIコーチ機能の提供・分析生成）

AIコーチ機能では、分析に必要な情報（業務内容、時間等）をAPIに送信しますが、個人を特定する氏名やメールアドレスは送信されません。また、送信されたデータがAIの学習に利用されない設定を適用しています。

4. 広告について

本アプリは Google AdMob による広告を表示します。iOS 14.5 以降では ATT に従い、利用者の許諾がある場合のみトラッキング識別子を利用します。許諾がない場合も、パーソナライズされない広告が表示されるのみで、アプリの全機能を利用可能です。

5. データの保存期間

  • 業務ログ・会話履歴：アカウント削除まで保持（Proプランは最大1年分を表示）
  • 認証情報：アカウント削除時に速やかに削除

6. アカウントおよびデータの削除

アプリ内「設定」→「アカウント削除」からいつでも退会・データ削除が可能です。削除要請は当方サポート窓口でも受け付けております。

7. 国外移転

本アプリのデータは、米国を含む日本国外のサーバー（Supabase等）に保存・処理される場合があります。当方は適切なデータ保護に努めます。

8. 改定

本ポリシーは必要に応じて改定することがあります。重要な変更はアプリ内または本ページにて告知します。
